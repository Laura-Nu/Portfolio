## Overview

INCOTERMS is an enterprise system built for **Soluciones Médicas Bilbao** to centralize and manage business-critical data such as products, imports, suppliers, and shippers. Its purpose is to optimize administrative processes, ensure secure storage of information, and streamline access to essential records for decision-making.

## Key Features

- **Product Registry**: Detailed cataloging of medical equipment with specifications and historical records.  
- **Supplier & Shipper Management**: Centralized storage of contacts and transaction history.  
- **Imports Tracking**: End-to-end management of import transactions from request to delivery.  
- **Centralized Database**: Built on Microsoft SQL Server 2022 for reliability and performance.  
- **Modular Architecture**: DAO-based design with clear separation between UI, business logic, and data access layers.  

## Technologies Used

- **ASP.NET Web Forms (ASPX)** for the presentation layer  
- **C# (ASPX.CS)** for business logic  
- **Microsoft SQL Server 2022** for database management  
- **HTML5, CSS, JavaScript** for the front-end UI  

## Challenges and Learnings

Implementing a layered architecture with DAO required careful structuring of interfaces, models, and implementations. Ensuring data security and maintaining consistent CRUD operations across multiple entities were central challenges. The team also gained experience with enterprise-grade database configuration and system hosting.

## Outcome

The project provided Soluciones Médicas Bilbao with a secure, scalable, and easy-to-use system. By consolidating operations into a single platform, the company improved efficiency in product cataloging, supplier management, and import tracking, supporting better control and decision-making in the medical equipment import sector.
