## Overview

SnapFlow is a web application for managing clothing sales and inventory through QR codes. It provides an easy-to-use platform for businesses to register products, track stock, and handle customer orders seamlessly.

## Key Features

- **QR-Based Inventory**: Each product is registered and tracked via unique QR codes.  
- **Company & User Management**: Businesses can manage multiple stores and employee accounts.  
- **Order Handling**: Integrated module for tracking and managing orders in real time.  
- **User Profile & Authentication**: Firebase-based secure login and profile management.  
- **Responsive Dashboard**: Sidebar navigation with role-based access.  

## Technologies Used

- **React.js**: Front-end development with modular components.  
- **Firebase**: Authentication, Firestore database, and storage.  
- **Bootstrap**: UI components for responsive design.  

## Challenges and Learnings

Synchronizing inventory with QR codes and real-time Firestore updates required careful handling of concurrency and database design.

## Outcome

SnapFlow streamlined clothing business operations by providing a modern, scalable, and intuitive system for managing sales and stock.
