## Overview

DocIA allows users to register health information and connect with biometric sensors (oxygen saturation, heart rate, temperature). It also provides family health monitoring, medical video calls, and a chatbot suggesting the most suitable hospital based on symptoms.

## Key Features

- **Biometric Sensor Integration**: Real-time data tracking.  
- **Conversational Chatbot**: Symptom-based hospital recommendations.  
- **Medical Video Calls**: Direct patient-doctor communication.  
- **Family Monitoring**: Shared dashboards for relatives.  

## Technologies Used

- **Flutter & Dart**  
- **Firebase (Realtime DB & Storage)**  

## Challenges and Learnings

The main challenge was integrating biometric devices with consistent real-time data flow, solved through Firebase’s real-time features.

## Outcome

DocIA became a comprehensive health platform, combining AI, telemedicine, and real-time monitoring.
