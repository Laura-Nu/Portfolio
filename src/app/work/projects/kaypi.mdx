## Overview

Kaypi is a mobile app that improves urban mobility by showing bus routes passing through selected map points, providing tourist information, and sending real-time alerts on disruptions.

## Key Features

- **Interactive Map**: Routes and stops visualization.  
- **Tourist Points**: Integration of key landmarks.  
- **Real-Time Alerts**: Notifications on incidents and blockages.  
- **Registered Lines Management**: Easy search and control of routes.  

## Technologies Used

- **Flutter & Dart**: Cross-platform mobile development.  
- **MongoDB**: Storage of transport and user data.  

## Challenges and Learnings

Integrating dynamic mapping and handling geospatial data required modular design and efficient queries.

## Outcome

The app increased accessibility of public transport information, improving mobility and user experience.
