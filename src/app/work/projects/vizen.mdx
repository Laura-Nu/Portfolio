## Overview

Vizen helps businesses estimate profitability timelines by automating financial projections. Users input sales prices, operational costs, and staff expenses to get sustainability insights.

## Key Features

- **Automated Accounting Projections**  
- **Clear Profitability Estimations**  
- **Business-Oriented Financial Modeling**  

## Technologies Used

- **TypeScript**  
- **Node.js**  
- **MySQL**  
- **Turborepo**  

## Challenges and Learnings

Designing accurate financial algorithms required validation with accounting experts.

## Outcome

Vizen accelerated financial analysis for SMEs, improving decision-making and clarity on economic sustainability.
